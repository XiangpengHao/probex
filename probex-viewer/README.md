# probex-viewer

Web-based trace viewer for [probex](https://github.com/XiangpengHao/probex).
Built with Dioxus and compiled to WebAssembly, it provides interactive timelines, flamegraphs, and IO/memory visualizations.
The viewer is embedded into the main `probex` binary and served locally after a profiling session.

See the main [probex](https://github.com/XiangpengHao/probex) repository for usage and documentation.
